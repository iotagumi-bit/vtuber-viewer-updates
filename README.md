# V-Tuber Model Viewer

Show your VTuber model right on your desktop. It receives the **Spout2** output from **VTube Studio** and displays it transparently on top of your screen, with aesthetic color filters.

*Made by **Torukana** — [youtube.com/@Torukana](https://www.youtube.com/@Torukana)*

---

## Before You Start

- **Windows 10/11**
- **VTube Studio** running, with **Spout2 Output enabled**

> Always launch VTube Studio and enable Spout2 **before** opening this app.

---

## Install

1. In VTube Studio, enable **Spout2 Output** in settings
2. Run **`VTuber Model Viewer Setup.exe`**
3. Open the app from your Desktop or Start Menu — your model appears automatically

## Lock Mode

When locked, the model becomes fully **click-through** — your mouse and clicks pass straight to the game behind it, with no accidental dragging or focus stealing. Great for gaming.

To unlock: **right-click the system tray icon → Unlock model**.

---

## Troubleshooting

**Model not showing / black screen**
Make sure VTube Studio is running with Spout2 enabled. Click ↻ Refresh, then ⏵ Connect.

**"This app can't run on your PC" from the Downloads folder**
Your Downloads folder may be OneDrive-synced (the file is only an online placeholder). Move the installer to another folder (e.g. drive D:) and run it.

**Antivirus removes/quarantines the app**
A common false-positive for unsigned apps. Restore it from quarantine and add it to your antivirus exclusions.
